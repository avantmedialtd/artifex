# Changelog

## 0.2.0 - Artifact-Aware Tree

- Artifact-aware tree hierarchy: changes show Proposal, Specs, Design, and Tasks nodes
- Artifact status inferred from file existence (no CLI dependency)
- Specs node expands to show individual capability specs
- Tasks node preserves section/checkbox drill-down with completion counts
- Click any present artifact to open its file
- Badge counts unfinished changes (including those without tasks.md)
- File watchers for all artifact types (design.md, specs/\*_/_.md)

## 0.1.0 - Initial Release

- Dedicated panel for monitoring OpenSpec changes
- Progress tracking with completion counts
- Badge notifications for active changes
- Real-time updates on file changes
- Click-to-navigate to task definitions
- Copy Change ID and Title commands
- Multi-root workspace support
- Auto-collapse completed sections setting
